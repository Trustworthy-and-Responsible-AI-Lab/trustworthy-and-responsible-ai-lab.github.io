No required textbook. Reading materials will be provided on the course website and/or distributed in class.


Prerequisites

This course requires a basic understanding of computer systems and computer security. Please consider taking these courses first:

  • CS 312 :: Linux System Administration (must)
  • CS 370 :: Introduction to Security (must)
  • CS 344 :: Operating Systems I (recommended)
  • CS 434 :: Machine Learning and Data Mining (recommended)
